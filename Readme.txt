						   -----> Инструкция по запуску бота <-----

						=====> Регистрация нового пользователя <=====

1) Запустите файл connection.exe
2) В появившемся окне введите номер телефону по указанному в программе примеру.
3) После того, как бот заполнит данные телефона и отправит запрос, перейдите в личный кабинет
   Wildberries и введите код для входа.
4) После того как вы зашли в свой профиль на сайте, не выходя из браузера, возвращайтесь в окно
   приложения, введите "yes" и нажмите "ENTER".
   Если при авторизации возникли трудности, то введите "no", выйдите из консоли
   и перезапустите файл connection.exe

							=====> Запуск Бота <=====

1) Запустите файл main.exe
2) Введите "1", если номер последнего пользователя (он будет отображен в приложении) совпадает с нужным для текущей сессии.
   Если вам нужно зайти под другим номером введите "2" и затем введите данные номера (ВНИМАНИЕ! Номер для входа обязательно должен
											быть зарегистрирован)
							    =====> Баги <=====

При багах и ошибках необходимо перезапустить бота. Если это не помогает - писать создателю. 
Например, сюда https://vk.com/dyupich.dyusha
						  =====> Изменение файла шаблона Excel<=====

Для того, чтобы изменить файл с шаблонами, необходимо перейти в каталог с файлами бота. Перейти в каталог ChromeDriver.
После этого удалить старый шаблон и добавить новый с именем "Шаблон.xlsx"
Внимание!
Не применяйте стили к пустым строкам в эксель-документах. Это может привести к некорректному считыванию размеров ячеек.
